Notes

  • There are two ways to write comment in javascript: // or /* */
  • A JavaScript program is made up of a series of statements. Each statement ends with a new line or semicolon
  • JavaScript has seven different data types. Six of them are primitive data types and are listed below: String, Symbol4, Number, Boolean, Undefined, Null. Any value that isnâ€™t one of the primitive data types listed above is an object.
  • typeof for finding out the type of a value. Example:
    typeof 'hello' << 'string'
    typeof 10 << 'number'
    typeof true << 'boolean'
    typeof { ninja: 'turtle' } << 'object'
    typeof [ 1, 2, 3 ] << 'object'
  • Variables have to be declared before they can be useD, and for that we use "let" or "const".
  • Scope:Using const and let to declare variables means they are block scoped, so their value only exists inside the block they are declared in.There are two:global scope and loca scope.
  • The slice() method creates a subarray; effectively chopping out a slice of an original array, starting at one position and finishing at another
    • The splice() method removes items from an array then inserts new items in their place.
  • A while loop and do ... while loop can be used to repeat a block of code while a condition is still true.
  • A for loop works in a similar way to a while loop, but has a different syntax
  • Parameters and arguments are terms that are often used interchangeably to represent values provided for the function as an input. There is a subtle difference though: any parameters a function needs are set when the function is defined. When a function is invoked, it is provided with arguments.
  • Function declarations can be invoked before they are defined because they are hoisted to the top of the scope, but function expressions cannot be invoked until after they are defined.
  • A callback is a function that is provided as an argument to another function.
